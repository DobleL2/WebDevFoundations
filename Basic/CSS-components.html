Componentes Básicos de CSS

Esta página explica los componentes básicos de CSS, con ejemplos prácticos para cada concepto.

1. Selectores

Los selectores se utilizan para seleccionar los elementos HTML a los que se aplicarán las reglas de estilo.

Ejemplo de un selector de clase:

.mi-clase {
    background-color: yellow;
}

Este es un ejemplo de un selector de clase.

Ejemplo de un selector de ID:

#mi-id {
    font-size: 20px;
    color: green;
}

Este es un ejemplo de un selector de ID.

2. Propiedades y Valores

Las propiedades en CSS definen qué aspecto de un elemento se va a cambiar, y los valores definen cómo se va a cambiar.

Ejemplo de cambiar el color del texto:

p {
    color: green;
}

Este texto es de color verde.

Ejemplo de cambiar el tamaño de la fuente:

p {
    font-size: 20px;
}

Este texto tiene un tamaño de fuente de 20px.

3. Modelo de Caja (Box Model)

El modelo de caja en CSS describe el modelo de diseño que utiliza CSS para ajustar los elementos en la página.

Este es un ejemplo de un modelo de caja.

Ejemplo de CSS para el modelo de caja:

div {
    width: 200px;
    padding: 10px;
    border: 2px solid black;
    margin: 20px;
}

4. Posicionamiento

CSS permite controlar cómo se posicionan los elementos en una página web.

Este es un ejemplo de posicionamiento relativo.

Ejemplo de CSS para posicionamiento relativo:

div {
    position: relative;
    top: 10px;
    left: 20px;
}

Flexbox es un sistema de diseño en CSS que permite crear diseños flexibles y eficientes.

6. Grid

CSS Grid es un sistema de diseño bidimensional que permite crear layouts complejos con filas y columnas.

Ejemplo de CSS Grid:

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

7. Media Queries

Las media queries se utilizan para aplicar estilos en función del tamaño de la pantalla u otras características del dispositivo.

Ejemplo de una media query:

@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}
